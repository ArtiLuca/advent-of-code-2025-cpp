# Advent of Code 2025 — C++ Solutions

This repository contains my personal solutions for
[Advent of Code 2025](https://adventofcode.com/2025).

The project is primarily a learning exercise. I am using the puzzles to practise:

- modern C++;
- problem decomposition;
- parsing structured input;
- algorithms and data structures;
- testing and debugging;
- analysing time and space complexity;
- organising small C++ programs with separate interfaces and implementations.

These are exploratory solutions rather than attempts to produce the shortest or
most competitive implementations. Some solutions may be revised as I discover
cleaner algorithms or improve my C++ design.
